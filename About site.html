About This Project

VoteSphere is a complete browser-based voting management system built with HTML, CSS, and JavaScript. It is designed for academic demos and prototype workflows where a lightweight setup is preferred.

Voter Features

Registration, login, open election discovery, one-click vote submission, and vote lock per election.

Admin Features

Create elections, set open or closed status, define candidates, and remove elections when needed.

Transparency

Live result calculation with winner summaries and candidate-wise vote counts.

Data is stored in your browser localStorage for demo purposes. This project can be extended to a backend service for production.
